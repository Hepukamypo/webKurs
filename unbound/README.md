## Быстрый старт

### 1. Создай базу данных PostgreSQL

Открой psql или pgAdmin и выполни:
```sql
CREATE DATABASE unbound_db;
CREATE USER unbound_user WITH PASSWORD 'yourpassword';
GRANT ALL PRIVILEGES ON DATABASE unbound_db TO unbound_user;
```

### 2. Настрой переменные окружения

Скопируй `.env.example` в `.env` и заполни:
```bash
cp .env.example .env
```
Открой `.env` и укажи свои данные:
```
SECRET_KEY=любая-длинная-случайная-строка
DEBUG=True
DB_NAME=unbound_db
DB_USER=unbound_user
DB_PASSWORD=yourpassword
DB_HOST=localhost
DB_PORT=5432
```

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

### 4. Применить миграции

```bash
python manage.py migrate
```

### 5. Создай суперпользователя (администратора)

```bash
python manage.py createsuperuser
```

После создания зайди в `/admin/` → Users → найди своего пользователя
→ поставь `role = admin`.

### 6. Запусти сервер

```bash
python manage.py runserver
```

Открой браузер: **http://127.0.0.1:8000**

## Функциональность

- **Каталог игр** — просмотр, фильтрация по жанру, сортировка
- **Страница игры** — обложка, скриншоты, достижения, отзывы
- **Аутентификация** — регистрация, вход, выход
- **Роли** — игрок и администратор с разными правами
- **CRUD** — администратор добавляет/редактирует/удаляет игры
- **Загрузка файлов** — обложки, скриншоты, дистрибутивы
- **Покупка и скачивание** — только купившие могут скачать
- **Достижения** — привязаны к играм, выдаются автоматически
- **Рейтинги** — средняя оценка из отзывов с агрегацией
- **Отзывы** — один отзыв на игру от пользователя
- **Друзья** — заявки, принятие/отклонение, список друзей
- **Профиль** — библиотека, достижения, очки, друзья
- **Admin-панель** — /admin/ для полного управления
